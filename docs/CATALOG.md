# CATALOG.md — Стратегия и структура каталога

> **Для Claude Code:** три представления — одна база товаров.  
> URL-параметр `?view=` управляет активным табом.

---

## АРХИТЕКТУРНЫЙ ПРИНЦИП

```
/catalog/?view=category  — по категории (по умолчанию)
/catalog/?view=segment   — по типу бизнеса
/catalog/?view=brand     — по производителю
```

Таб-переключатель — постоянный элемент страницы `/catalog/`.  
При смене таба → меняется только содержимое ниже, URL обновляется через `router.push`.

## ПРЕДСТАВЛЕНИЕ 2 · ПО ТИПУ БИЗНЕСА

Каждая страница сегмента — мини-лендинг для конкретной аудитории.

### Сегменты

| Сегмент | slug | H1 |
|---|---|---|
| Пекарня и кулинария | bakery-shop | Ингредиенты для пекарни — надёжно, технологично, с поддержкой |
| Кондитерская | confectionery-shop | Ингредиенты для кондитерской — от Barry Callebaut до фруктовых пюре |
| Ресторан и кафе | restaurant | Ингредиенты для ресторана — полный ассортимент для любой кухни |
| Кофейня | coffee-shop | Всё для кофейни — от альтернативного молока до топпингов |
| Бургерная и фастфуд | burger | Ингредиенты для бургерной — стабильность в каждой смене |
| Столовая и корппит | canteen | Ингредиенты для столовой — объём, качество, предсказуемость |
| Пищевое производство | production | Ингредиенты для производства — оптовые поставки под ваши задачи |

### Типичные задачи по сегментам

**Пекарня:**
- Нужны стабильные дрожжи с предсказуемым подъёмом
- Ищем готовую смесь под новую позицию меню
- Хотим снизить себестоимость без потери качества
- Нужна помощь с разработкой рецептуры

**Кондитерская:**
- Ищем шоколад с нужным процентом какао для покрытия
- Нужны начинки с долгим сроком годности
- Хотим обновить линейку тортов — нужна идея и ингредиент
- Ищем альтернативу текущему поставщику сливок

**Кофейня:**
- Ищем альтернативное молоко для latte art
- Нужны базы для фирменных напитков
- Хотим добавить горячий шоколад в меню
- Ищем поставщика сиропов и топпингов

**Бургерная:**
- Хлебобулочные изделия для бургеров (смеси, стабилизаторы)
- Соусы и майонезы оптом
- Масло для фритюра с высокой точкой дымления
- Упаковка для навынос

### Структура страницы сегмента

```
[HERO с фото и заголовком под аудиторию]
  border-left: 3px solid [L1 сегмента]

[ТИПИЧНЫЕ ЗАДАЧИ — 4 карточки]
  border-top: [L1 сегмента]
  каждая ведёт в нужную категорию

[ЛИСТИНГ ТОВАРОВ]

  фильтры: производитель, объём, категория

  Warm фон, Amber акцентная линия

[ОТЗЫВ КЛИЕНТА ИЗ СЕГМЕНТА]
  из таблицы Testimonial где segment = slug
```

## SEO-АРХИТЕКТУРА

### Кластеры запросов

**Кластер 1 — Категорийные (транзакционные):**
```
ингредиенты для пекарни оптом Самара
хлебопекарные смеси купить Поволжье
ингредиенты для кондитерской оптом
поставщик ингредиентов HORECA Самара
```

**Кластер 2 — Брендовые:**
```
Barry Callebaut дистрибьютор Самара
дрожжи Lesaffre Саф-Нева купить Поволжье
Backaldrin ингредиенты Самара
```

**Кластер 3 — Сегментные:**
```
ингредиенты для пекарни Самара
ингредиенты для кофейни оптом
поставщик для ресторана Поволжье
```

### URL-структура

```
/catalog/                          → входная (три таба)
/catalog/category/bakery/          → хлебопекарное
/catalog/category/chocolate/       → шоколад
/catalog/segment/bakery-shop/      → для пекарни
/catalog/segment/coffee-shop/      → для кофейни
/catalog/brands/                   → все производители
/catalog/brands/barry-callebaut/   → страница бренда
/catalog/products/[slug]/          → карточка товара
```

### Метатеги

```typescript
// Категория
title: `${category.name} оптом в Самаре | ПАНЭМ`
description: `${category.description}. Доставка по Поволжью. Технологическая поддержка 24/7.`

// Сегмент
title: `Ингредиенты для ${segment.name} оптом | ПАНЭМ Самара`

// Бренд
title: `${brand.name} в Самаре — официальный дистрибьютор ПАНЭМ`
description: `Полный ассортимент ${brand.name} в наличии на складе в Самаре.`

// Товар
title: `${product.name} ${brand.name} | ПАНЭМ`
description: `${product.description}. Фасовка: ${product.packaging}.`
```
